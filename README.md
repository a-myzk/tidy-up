# README

## 開発言語
- Ruby 2.6.5
- Ruby on Rails 5.2.5

## 就業Termの技術
- Ajax
- お気に入り機能（フラグ機能）

## カリキュラム外の技術
- chartkick

## 実行手順
```
$ git clone https://github.com/a-myzk/tidy-up.git
$ cd tidy-up
$ bundle install
$ rails db:create db:migrate
$ rails s
```
